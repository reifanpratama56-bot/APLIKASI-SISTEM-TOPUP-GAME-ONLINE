Ini adalah halaman paling seru dan paling rumit. Di sinilah pengguna bisa membeli diamond untuk game favorit mereka. Bayangkan seperti berbelanja di toko game online.

Langkah 1: Pilih Game
Pengguna akan melihat 16 game populer yang tersusun dalam grid 2 baris x 8 kolom. Setiap game ditampilkan dalam sebuah kartu kecil berisi:

Logo game (gambar ikon game)

Nama singkat game (misal: "MLBB" untuk Mobile Legends)

Publisher game (misal: "Moonton")

Ada juga kotak pencarian di bagian atas. Pengguna bisa mengetik nama game untuk mencari lebih cepat.

Saat pengguna mengklik sebuah game:

Kartu game berubah background menjadi biru muda

Border kartu berubah menjadi biru tebal

Sistem memuat daftar nominal untuk game tersebut

Langkah 2: Pilih Nominal
Setelah game dipilih, muncul pilihan nominal di bawahnya. Nominal ini berbeda-beda untuk setiap game. Contoh:

Mobile Legends:

86 Diamonds = Rp 18.000

172 Diamonds = Rp 35.000

257 Diamonds = Rp 52.000

344 Diamonds = Rp 69.000

429 Diamonds = Rp 86.000

514 Diamonds = Rp 103.000

706 Diamonds = Rp 141.000

963 Diamonds = Rp 192.000

1412 Diamonds = Rp 282.000

2195 Diamonds = Rp 439.000

Free Fire:

100 Diamonds = Rp 15.000

210 Diamonds = Rp 30.000

355 Diamonds = Rp 50.000

520 Diamonds = Rp 73.000

720 Diamonds = Rp 100.000

1060 Diamonds = Rp 147.000

2180 Diamonds = Rp 300.000

Setiap kartu nominal menampilkan:

Jumlah (misal: "86")

Jenis item (misal: "💎 Diamond")

Harga (misal: "Rp 18.000")

Sistem secara otomatis mendeteksi jenis item dari nama nominalnya:

"Diamond" → 💎 Diamond

"CP" → 🔫 CP (COD Points)

"UC" → 🎯 UC (Unknown Cash)

"Robux" → 🤖 Robux

Dan seterusnya

Langkah 3: Pilih Metode Pembayaran
Ada 4 pilihan E-Wallet (dompet digital):

Dana (biaya admin Rp 2.500)

GoPay (biaya admin Rp 2.000)

Ovo (biaya admin Rp 1.500)

ShopeePay (biaya admin Rp 2.000 + diskon 5%!)

ShopeePay adalah pilihan paling hemat karena ada diskon 5%.

Langkah 4: Isi Data Pembeli
Pengguna harus mengisi:

ID GAME - ID akun game (misal: ID Mobile Legends)

EMAIL - Alamat email untuk konfirmasi

KODE PROMO - Jika punya kode promo, masukkan di sini

Cara Menggunakan Kode Promo
Ada dua kode promo yang tersedia:

WEEKEND20 - Diskon 20% dari total harga

FREEADMIN - Gratis biaya admin

Cara pakai:

Ketik kode promo di kotak yang tersedia

Klik tombol "APPLY"

Jika kode benar, diskon langsung terlihat di ringkasan

Langkah 5: Lihat Ringkasan Pembayaran
Di bagian kanan bawah, ada ringkasan yang menunjukkan:

Game yang dipilih

Nominal yang dipilih

Harga Game

Biaya Admin

Metode Bayar

Promo (jika ada)

TOTAL BAYAR (dalam huruf besar dan warna biru)

Ada juga progress bar yang menunjukkan kelengkapan data:

25% = "Lengkapi data" (belum semua data terisi)

75% = "Siap bayar!" (semua data sudah lengkap)

Langkah 6: Proses Pembayaran
Setelah semua data lengkap, pengguna bisa klik tombol "BAYAR SEKARANG". Lalu terjadi proses:

1. Validasi

ID Game tidak boleh kosong

Email harus valid (mengandung karakter @)

2. Muncul Dialog Konfirmasi

Dialog ini berisi:

Countdown Timer 10 menit - Pengguna harus menyelesaikan pembayaran sebelum waktu habis

Nomor Virtual Account - Nomor unik untuk pembayaran, misal "88101781837395"

Tombol Salin VA - Untuk menyalin nomor VA ke clipboard

Struk Pembelian - Detail transaksi

Instruksi Pembayaran - Panduan langkah demi langkah sesuai E-Wallet yang dipilih

Contoh Instruksi untuk Dana:

Buka aplikasi DANA

Pilih menu 'Bayar' atau 'Transfer'

Pilih 'Virtual Account' / 'VA'

Masukkan nomor VA: 88101781837395

Masukkan nominal: Rp 18.000

Konfirmasi dan selesaikan pembayaran

3. Animasi Loading

Setelah pengguna mengklik "Saya Sudah Bayar", muncul dialog dengan progress bar:

0-20%: "Memverifikasi pembayaran..."

20-40%: "Menghubungi server game..."

40-60%: "Mengirim diamond..."

60-80%: "Topup sedang diproses..."

80-95%: "Hampir selesai..."

95-100%: "Topup berhasil!"

4. Dialog Sukses

Setelah loading selesai, muncul struk sukses dengan:

Kode Transaksi (misal: TRX1781837395843A7B)

Detail transaksi (Game, Nominal, Metode, Total)

ID Game yang di-topup

Poin yang didapat

Tombol "OK"

Setelah klik OK, pengguna ditanya: "Ingin melakukan topup lagi?"

Jika "Ya" → Form direset, siap untuk topup baru

Jika "Tidak" → Kembali ke dashboard

Fitur Reset Form
Tombol "RESET FORM" berguna jika pengguna ingin memulai dari awal:

Menghapus semua pilihan (game, nominal, metode)

Mengosongkan field input (ID Game, Email, Kode Promo)

Mereset progress bar

Menampilkan notifikasi "Form berhasil direset!"
