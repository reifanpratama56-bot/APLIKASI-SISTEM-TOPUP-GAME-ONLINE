Ini adalah "kantor pusat" aplikasi. Setelah login, pengguna akan melihat dashboard yang berisi semua informasi penting dan tombol navigasi ke fitur-fitur lain.

Desain Dashboard
Dashboard menggunakan tema warna biru dengan gradasi dari biru terang ke biru tua. Semua komponen tersusun rapi di tengah layar dengan background biru yang menenangkan.

Bagian-bagian Dashboard
1. Header (Bagian Atas)
Di sini ada:

Avatar berbentuk emoji 👤

Sapaan "Halo, [nama pengguna]" misalnya "Halo, reifan"

Status "Member sejak 2026" (menunjukkan pengguna sudah terdaftar)

2. Tiga Kartu Statistik
Tiga kartu berisi informasi penting pengguna:

Kartu 1: ⭐ POIN SAYA
Menampilkan jumlah poin yang dimiliki pengguna. Poin ini didapat dari setiap transaksi topup. Setiap Rp 10.000 = 1 poin.

Kartu 2: 💰 TOTAL TOPUP
Menampilkan total uang yang sudah dihabiskan pengguna untuk topup, misalnya "Rp 1.000.000".

Kartu 3: 🎮 GAME FAVORIT
Menampilkan game yang paling sering di-topup oleh pengguna. Sistem menghitung ini secara otomatis dari riwayat transaksi. Misalnya jika pengguna sering topup Mobile Legends, maka akan muncul "Mobile Legends".

3. Promo Hari Ini
Menampilkan daftar promo yang sedang aktif:

🎁 WEEKEND20 - Diskon 20% semua game!

🎉 FREEADMIN - Gratis biaya admin!

4. Empat Tombol Navigasi
Tombol-tombol ini adalah jalan untuk berpindah ke fitur lain:

Tombol 1: 🎮 TOPUP
Untuk melakukan pembelian diamond atau item game. Ini adalah fitur utama aplikasi.

Tombol 2: 📋 HISTORY
Untuk melihat semua riwayat transaksi yang pernah dilakukan.

Tombol 3: 👤 PROFILE
Untuk melihat data diri pengguna (username, email, no HP, poin, total topup).

Tombol 4: 🚪 LOGOUT
Untuk keluar dari akun. Akan muncul konfirmasi "Yakin ingin logout?" sebelum benar-benar keluar.

Cara Kerja Perhitungan Game Favorit
Sistem melakukan hal berikut untuk menentukan game favorit:

Ambil semua transaksi milik pengguna

Hitung berapa kali setiap game muncul

Pilih game dengan jumlah terbanyak

Jika belum ada transaksi, tampilkan "Belum ada topup"
